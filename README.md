# vue-til

## 기술 스택
- Vue.js 2.6.10
- Vue Router 4.0.13
- Vuex 4.0.2
- Axios
- Vue Test Utils

## 뷰 라우터 및 컴포넌트 설계
- router-link 및 router-view 이용
- 코드 스플리팅 component: () => import('경로')
- 주요 페이지 경로 main, login, signUp, post/:id

## 토큰관리
- JSON Web Token
- Authorization 토큰 값으로 API 인증을 받는 방법
- axios.interceptors
- cookie를 이용한 로그인 인증 값 저장
